# Security Policy

## Supported Versions

Only 1.9.x is supported with bug fixes and security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.9.x   | :white_check_mark: |
| < 1.9.x | :x:                |

I will reply/respond within 24 hours (due to diffrent time zones)
